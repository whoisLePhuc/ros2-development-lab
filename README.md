# ROS2 Zero to Hire 🚀

Lộ trình học ROS2 từ cơ bản đến chuyên nghiệp — dành cho người mới bắt đầu.

## Cấu trúc

```
ROS2-Zero-to-Hire/
├── NN.ten-bai/
│   ├── ten-bai.md          # Lý thuyết + hướng dẫn
│   ├── Exercises/          # Bài tập thực hành
│   └── code/               # Mã nguồn ví dụ
├── assets/                 # Hình ảnh, tài nguyên
├── docs/                   # Tài liệu hướng dẫn chung
├── README.md
└── ROADMAP.md
```

## Yêu cầu

- **Ubuntu 22.04** (Jammy)
- **ROS2 Humble** (Hawksbill)
- Python 3.10+
- C++17 (tùy chọn, học sau)

## Cách học

1. Đọc file `.md` trong từng thư mục bài học
2. Làm bài tập trong `Exercises/`
3. Xem code mẫu trong `code/`
4. Mỗi bài có link "Bài tiếp theo" ở cuối
